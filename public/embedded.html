Embedded Chat Widget Example

Chat widget embedded within page content

About This Demo

This page demonstrates how to embed the Flowise chat widget within a specific container on your webpage, alongside other content.

The chat widget is initialized using Chatbot.initFull() and targeted to a specific HTML element using the id parameter.

Features

  • Embedded within page layout
  • Custom container sizing
  • Responsive design
  • Configurable welcome message
  • Starter prompts enabled
  • Auto-message on load

Use Cases

This mode is ideal for:

  • Support pages
  • Help centers
  • Product documentation
  • Customer service portals
